# Python Crypto Trading BOT

This is a Python-based cryptocurrency trading bot that interacts with the Binance exchange. It allows users to automate their trading strategies using the BinanceTH API.

## API Documentation
https://www.binance.th/api-docs/en/#rest-open-api

## Features
- Connects to Binance exchange using API keys.
- Fetches account information and market data.
- Places buy and sell orders based on predefined strategies.
- Monitors open orders and trade history.

## Requirements
- Python 3.7+

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kokomyoheinhtet/crypto-trading-bot.git
    ```
2. Navigate to the project directory:
3. ```bash
   cd crypto-trading-bot
   ```
4. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
5. Set up your Binance API keys in the `config.py` file.
6. Run the bot:
   ```bash
   python run_bot.py
   -- or
   python run_bot_pro.py
   ```
